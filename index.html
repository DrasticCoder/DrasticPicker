Welcome to DrasticPicker

DrasticPicker is a Yet Another Color Picker Extension neobrutalism–themed Chrome extension for picking colors from any webpage. With a powerful zoom lens, crosshair cursor, and recent color history, picking the perfect color has never been so bold!

Features

  • Full-page picker overlay with a zoom lens
  • Crosshair cursor for precision
  • Keyboard shortcuts for quick access (Alt+P to open picker, Ctrl+C to copy current color)
  • Recent color history (last 5 colors) for fast reuse
  • Neobrutalism design – bold, raw, and unapologetic

Installation

  1. Download the ZIP file above.
  2. Open Chrome and navigate to chrome://extensions/.
  3. Enable Developer Mode.
  4. Click Load unpacked and select the extension folder.
  5. Use Alt+P or click the extension icon to launch the color picker.
